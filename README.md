# Book Tracker

**Автор:** Михаил Эрастов

## Описание программы

Book Tracker — простое приложение для учёта прочитанных книг. Позволяет добавлять книги с указанием названия, автора, жанра и количества страниц. Есть фильтрация по жанру и количеству страниц. Данные сохраняются и загружаются из файла JSON.

## Инструкция по запуску

1. Установите Python 3.x.
2. Установите зависимости (если есть).
3. Запустите файл `book_tracker.py`.

## Примеры использования

- Добавьте книгу через форму и нажмите «Добавить книгу».
- Используйте фильтрацию по жанру или количеству страниц.
- Сохраните данные кнопкой «Сохранить», загрузите — «Загрузить».
